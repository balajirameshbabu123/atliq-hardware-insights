# atliq-hardware-insights
![Atliq.png](Atliq.png)
Project Description :
As a data analyst in this project, my main responsibility is to analyze the sales data of AtliQ Hardware stored in MySQL database, unlock sales insights and also build a Power BI dashboard to automate the insights to reduce manual time spent on data gathering and make it available to the sales team for making decisions.

Processes Involved:
- Planning the project using AIMS Grid & Data Discovery.
-Drew insights of the data like sales, revenue, transactions, markets etc using MySQL to have a general overview of the project.
-Performed ETL process and extracted data from MySQL and loaded it into Power BI for analysis and reporting.
- Using Power Query Editor performed data cleaning, added new measures using DAX, and included new relevant columns to enhance the quality of the data and improve the analysis in Power BI.
Insights :
-Revenue and quantity by customer, date and year.
-Cost and Revenue by region or market.
-Top 5 customers and products.
-Bottom 5 customers and products.
- Profit margin % per market, Profit margin contribution % per market.
- Revenue % by market, Revenue contribution % by market.
-Sales and Profit margin contribution % by zones and created red alerts (through conditional formatting) to understand which markets in the zones( North, South, West) are doing good and which are not.
-Breakdown of sales for brick and motor vs e-commerce customers.
- Comparison between last year's and present year's revenue at the same time.

Result: Designed an automated Power BI dashboard to understand Atliq hardware goods and E-commerce sales trends. The stakeholders gathered together, created a combined log of the comments and consolidated them to identify the different levels of root causes and made some data-driven business decisions which could help in increasing the revenue at least by 7% in the next quarter, with the help of this dashboard.
